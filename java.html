What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). Released in 1995, Java follows the "Write Once, Run Anywhere" (WORA) principle, meaning compiled Java code can run on any platform that has a Java Virtual Machine (JVM).

Why learn Java? Java is widely used in enterprise applications, Android app development, web applications, and large-scale systems. It's known for its robustness, security, and extensive ecosystem.

Installation & Setup

Follow these steps to set up Java on your computer:

  1. Download JDK (Java Development Kit) from oracle.com (JDK 17 or later recommended)
  2. Install JDK and set JAVA_HOME environment variable
  3. Verify installation: Open terminal/command prompt and type: java -version
  4. Install an IDE: IntelliJ IDEA (recommended) or VS Code with Java extensions
  5. Configure your IDE to use the installed JDK

Your First Java Program

Let's write your first Java program! Create a file called HelloWorld.java and type the following:

Java 
// Your first Java program
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("Welcome to Java programming!");
        
        // Variables
        String name = "CodePath";
        int age = 30;
        System.out.println("My name is " + name + " and I'm " + age + " years old.");
    }
}

Compile with javac HelloWorld.java and run with java HelloWorld. You should see the output!

Example: Classes and Objects

Java 
// Class definition
public class Student {
    // Private attributes (Encapsulation)
    private String name;
    private int age;
    
    // Constructor
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Getter methods
    public String getName() {
        return name;
    }
    
    public int getAge() {
        return age;
    }
    
    // Method
    public void displayInfo() {
        System.out.println("Name: " + name + ", Age: " + age);
    }
}

// Main class
public class Main {
    public static void main(String[] args) {
        // Create objects
        Student student1 = new Student("Alice", 20);
        Student student2 = new Student("Bob", 22);
        
        student1.displayInfo();
        student2.displayInfo();
    }
}
