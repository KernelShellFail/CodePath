What are C & C++?

C is a powerful, low-level programming language developed in the 1970s. It's the foundation of many modern languages and is known for its performance and direct memory access.

C++ is an extension of C with object-oriented programming features. Created by Bjarne Stroustrup, C++ combines the power of C with modern OOP concepts, making it ideal for system programming, game development, and performance-critical applications.

Installation & Setup

Follow these steps to set up C/C++ on your computer:

  1. Windows: Install MinGW or Visual Studio with C++ tools
  2. macOS: Install Xcode Command Line Tools: xcode-select --install
  3. Linux: Install GCC: sudo apt install gcc g++
  4. Install a code editor: VS Code (recommended) with C/C++ extensions
  5. Verify installation: gcc --version and g++ --version

Your First C Program

Let's write your first C program! Create a file called hello.c:

C 
// Your first C program
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("Welcome to C programming!\n");
    
    // Variables
    int age = 30;
    float price = 19.99;
    char grade = 'A';
    
    printf("Age: %d, Price: %.2f, Grade: %c\n", age, price, grade);
    return 0;
}

Compile with gcc hello.c -o hello and run with ./hello (Linux/Mac) or hello.exe (Windows).

Example: Classes and Objects in C++

C++ 
#include <iostream>
using namespace std;

class Student {
private:
    string name;
    int age;

public:
    // Constructor
    Student(string n, int a) {
        name = n;
        age = a;
    }
    
    // Member function
    void displayInfo() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
};

int main() {
    // Create objects
    Student student1("Alice", 20);
    Student student2("Bob", 22);
    
    student1.displayInfo();
    student2.displayInfo();
    
    return 0;
}
