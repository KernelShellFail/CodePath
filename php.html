üöÄ Getting Started with PHP

What is PHP?

PHP (Hypertext Preprocessor) is a server-side scripting language designed for web development. Created by Rasmus Lerdorf in 1994, PHP is one of the most popular backend languages, powering millions of websites including Facebook, WordPress, and Wikipedia.

Why learn PHP? PHP is easy to learn, has excellent database support (especially MySQL), and is widely used for building dynamic websites, content management systems, and e-commerce platforms.

Installation & Setup

Follow these steps to set up PHP on your computer:

  1. Windows: Install XAMPP or WAMP (includes PHP, Apache, MySQL)
  2. macOS: Install XAMPP or use Homebrew: brew install php
  3. Linux: Install LAMP stack: sudo apt install php apache2 mysql-server
  4. Verify installation: Open terminal and type: php -v
  5. Install a code editor: VS Code with PHP extensions

Your First PHP Program

Let's write your first PHP program! Create a file called hello.php in your web server directory (htdocs for XAMPP):

PHP 
<?php
// Your first PHP program
echo "Hello, World!";
echo "<br>Welcome to PHP programming!";

// Variables
$name = "CodePath";
$age = 30;
echo "<br>My name is " . $name . " and I'm " . $age . " years old.";

// Output with echo
echo "<br>PHP is awesome!";
?>

Access it in your browser: http://localhost/hello.php (make sure Apache is running in XAMPP/WAMP).

Example: Handling Forms with $_POST

PHP 
<?php
// form_handler.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = $_POST['name'];
    $email = $_POST['email'];
    
    echo "Hello, " . htmlspecialchars($name) . "!<br>";
    echo "Your email is: " . htmlspecialchars($email);
}
?>
