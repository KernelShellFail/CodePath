üöÄ Getting Started with Python

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability and allows developers to express concepts in fewer lines of code than languages like C++ or Java.

Why learn Python? Python is widely used in web development, data science, artificial intelligence, automation, scientific computing, and more. It's beginner-friendly yet powerful enough for professional applications.

Installation & Setup

Follow these steps to set up Python on your computer:

  1. Download Python from python.org (Python 3.11 or later recommended)
  2. Run the installer and check "Add Python to PATH"
  3. Verify installation by opening terminal/command prompt and typing: python --version
  4. Install a code editor: VS Code (recommended) or PyCharm
  5. Install Python extension in VS Code

Your First Python Program

Let's write your first Python program! Create a file called hello.py and type the following:

Python 
# Your first Python program
print("Hello, World!")
print("Welcome to Python programming!")

# Variables
name = "CodePath"
age = 30
print(f"My name is {name} and I'm {age} years old.")

Run this program by typing python hello.py in your terminal. You should see the output displayed on your screen!

# Conditional Statements

    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"Score: {score}, Grade: {grade}")

Example: Lists and Loops

Python 
# Lists and Loops
fruits = ["apple", "banana", "cherry", "date"]

# Using for loop
for fruit in fruits:
    print(f"I like {fruit}")

# Using while loop
count = 0
while count < len(fruits):
    print(f"{count + 1}. {fruits[count]}")
    count += 1

# List comprehension
numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
print(squares)  # [1, 4, 9, 16, 25]
